## 🎥 OBS Studio Integration

### Quick Setup for Streamers

1. **Open OBS Studio**

2. **Add Browser Source:**
   - Sources → `+` → Browser
   - Name: `Event Pulse Dashboard`

3. **Configure Settings:**
   ```
   URL: https://samotech.github.io/EventPulseChat/
   Width: 1920
   Height: 1080
   FPS: 30
   ```

4. **Recommended Settings:**
   - ✅ Shutdown source when not visible
   - ✅ Refresh browser when scene becomes active
   - ❌ Control audio via OBS (leave unchecked)

5. **Optional: Transparent Background**
   ```css
   body { background: transparent !important; }
   ```

## 🎨 Customization

### Modify Sports Coverage

Edit `script-simple.js` to add/remove sports:

```javascript
// Change sport leagues
case 'soccer':
    data = await fetchESPNScores('soccer/ita.1'); // Serie A instead
    break;

case 'nba':
    data = await fetchESPNScores('basketball/nba');
    break;
```

### Adjust Refresh Rate

```javascript
// Default: 30 seconds
setInterval(async () => {
    await loadLiveScores(activeSport);
    renderGames(activeSport);
}, 30000); // Change to 15000 for 15 seconds
```

### Add Custom Sport

1. **Add Tab Button** (index.html):
```html
<button class="tab-button" data-sport="formula1">
    <span class="tab-icon">🏎️</span>
    <span class="tab-label">F1</span>
</button>
```

2. **Add Content Section** (index.html):
```html
<div class="tab-content" id="formula1">
    <div class="sport-header">
        <h2>🏎️ Formula 1</h2>
    </div>
    <div class="games-grid" id="formula1-games"></div>
</div>
```

3. **Add API Handler** (script-simple.js):
```javascript
case 'formula1':
    data = await fetchESPNScores('racing/f1');
    break;
```

### Style Customization

**Colors** (style.css):
```css
/* Change primary colors */
.header {
    background: linear-gradient(90deg, #YOUR_COLOR 0%, #YOUR_COLOR 100%);
}

.game-card {
    border: 3px solid #YOUR_COLOR;
}
```

## 🐛 Troubleshooting

### Dashboard Not Loading

1. **Check GitHub Pages Status:**
   - Go to Settings → Pages
   - Should show green box: "Your site is live at..."

2. **Wait for Deployment:**
   - GitHub Pages needs 2-3 minutes to build
   - Try refreshing after waiting

3. **Hard Refresh Browser:**
   - Windows/Linux: `Ctrl + Shift + R`
   - Mac: `Cmd + Shift + R`

### Scores Not Updating

1. **Open Console (F12):**
   - Should see: `🔴 Loading LIVE scores...`
   - Should see: `✅ Loaded X games`

2. **Check Network Tab:**
   - Look for ESPN API calls
   - Status should be 200 OK

3. **Verify Live Games:**
   - Some sports may not have games today
   - Check ESPN.com for schedule

### Modal Not Opening

1. **Check Console for Errors:**
   - Press F12 → Console tab
   - Look for JavaScript errors

2. **Clear Browser Cache:**
   - Settings → Clear browsing data
   - Reload page

3. **Try Different Browser:**
   - Test in Chrome/Firefox

### OBS Source Issues

1. **Browser Source Settings:**
   - Width: 1920
   - Height: 1080
   - FPS: 30

2. **Refresh Source:**
   - Right-click source → Refresh

3. **Check URL:**
   - Must be `https://` not `http://`
